## go TCP网络框架封装学习
### 概述
 近期学习golang的网络编程,找到一个不错的视频  [zinx](https://www.bilibili.com/video/BV1wE411d7th?p=29)<br>
 该视频是从封装一个tcp网络框架开始,连接,消息,请求,路由,消息处理等由浅入深,<br>
 不断迭代框架.框架封装完成后,使用框架做游戏服务器开发.
